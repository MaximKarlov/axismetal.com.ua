Опис: комплект поздовжньої різки м’якої плівки призначений для розрізання плівки по довжині запакованого лотка в термоформувальних машинах. Він включає в себе 5 дискових ножів на втулках, вал для дискових ножів, куди вони кріпляться для того, щоб можна було змінювати формат запаювання лотків та роги, які…

Застосування: різка м’якої плівки по довжині на термоформері.

Сумісність: Термоформер ULMA, Webomatic, Multivac

Ми пропонуємо виготовлення для вас ножів, прес-форм, комплектів різки для трейсилерів та комплектів формовки та запайки, матриць для термоформерів по індивідуальному замовленню, в будь-якому форматі та з будь-якими складовими комплекту. Все, що потрібно - це зателефонувати нашому менеджеру та озвучити технічне завдання. Або прикріпити та відправити нам креслення через спеціальну форму на сайті.

В комплект входить:

Вал для дискових ножів поздовжньої різки, 653х35мм, на термоформер, для розширення формату запайки (Україна, АХІSMETAL)

Опис: Вал для дискових ножів поздовжньої різки на термоформер, довжиною 653 мм та висотою 35 мм по центру вала, виготовлений з високоякісної сталі, що використовується для розширення варіантів формату запечатування. Наприклад, виробництво пакувало по 2 лотки за такт, а потрібно перейти на 6.

Застосування: Для розширення формату запайки

Сумісність: термоформери ULMA, Webomatic, Multivac
